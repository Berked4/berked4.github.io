# Diri Games — static website

The Diri Games static website. Hosts the Bouncy Pop game page, support/contact
page, and privacy policy.

- Static HTML/CSS only — no build tools, no JavaScript, no external assets.
- Intended for hosting on GitHub Pages.

## Pages

- `/` — Diri Games home
- `/bouncy-pop/` — Bouncy Pop game page
- `/bouncy-pop/privacy/` — Bouncy Pop privacy policy
- `/support/` — support and contact
